IMPORTANT
=========
- All statusbar scripts and are in scripts folder. Please move whatever you want to use to your $PATH variable. (Note that dwm scripts are in dotfiles repo, as well as all of these)
- Checkout .cfg files in statusbar for how to make/add your own files. It is pretty easy, just be sure not to use same signal for different blocks.
- Remember to add/delete your included blocks from "dwmblocks.cfg", which is the global file for blocks.
